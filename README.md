# sp-to-ps-c

This plugin is modified from the official plugin, mainly adding the functionality to automatically generate solid color layers in Photoshop.

## Installation

Copy the `sp-to-ps-c` folder to `Documents\Adobe\Adobe Substance 3D Painter\javascript\plugins`.
